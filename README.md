Esta lista de exercícios visa consolidar os conceitos básicos de programação usando a linguagem Python . O foco principal é a prática de manipulação de variáveis, tipos de dados (inteiros, reais e strings), operadores aritméticos e as funções fundamentais de interação com o input()usuário print().

🎯 Objetivos de Aprendizagem
Interação do usuário: Aprenda a exibir mensagens e capturar dados inseridos no teclado.

Manipulação de variáveis: Armazenamento de informações de diferentes tipos para uso posterior.

Lógica Matemática: Aplicar fórmulas aritméticas para resolver problemas do dia a dia, como conversão de unidades e cálculos de área.

Formatação da saída: Exibir os resultados de forma clara e organizada.

📋 Exercício Roteiro
A atividade está dividida em desafios progressivos:

Saída simples: Exibição da mensagem "Olá terráqueos!!".

Entrada e Saída: Capture um número e exiba mensagens personalizadas.

Operações aritméticas básicas: Soma de valores e cálculo da média aritmética.

Conversão de Medidas: Transformação de metros em centímetros.

Geometria básica: Cálculo da área de círculos e quadrados.

Cálculos financeiros: Processamento salarial com base nas horas trabalhadas.

Termodinâmica: Conversão mútua entre as escalas Celsius e Fahrenheit usando fórmulas específicas.

Expressões complexas: exercícios de lógica matemática envolvendo potências, produtos e manipulação de múltiplos tipos numéricos.

Fórmulas biométricas: Cálculo do peso ideal com base na altura.
