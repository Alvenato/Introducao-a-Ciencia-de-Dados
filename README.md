📊 Atividades práticas: Fundamentos de programação para ciência de dados

Esta lista de exercícios compõe o módulo inicial de Introdução à Ciência dos Dados . O objetivo é desenvolver fluência em Python, tratando a linguagem não apenas como um programa, mas como uma ferramenta para processar sinais e dados brutos .

Nesta etapa, focamos na construção de scripts que simulam o fluxo de entrada, transformação e saída de informações — com base em qualquer pipeline de dados.

🧬 Objetivos Analíticos
Entrada e análise de dados: Pratique a captura de dados externos e a conversão de tipos (casting) para garantir a integridade de dois cálculos.

Engenharia de Recursos (Básico): Criar novas métricas a partir de variáveis ​​existentes (ex: converter unidades e calcular índices como o peso ideal).

Lógica Algorítmica: Aplicar fórmulas matemáticas complexas para extrair conhecimento de variáveis ​​numéricas.

Interface de Resultados: Comunicar as conclusões de forma clara e amigável é uma habilidade essencial para o cientista de dados.

🔍 Estrutura de Dois Desafios
A lista está organizada em três pilares do pensamento computacional aplicados aos anos em questão:

Manipulação de Variáveis ​​​​e Typagem: Exercícios de saudação e eco de dados para compreensão do comportamento de strings e personas.

Operações estatísticas e geométricas: cálculo de médias, áreas e dados, fundamentais para a análise descritiva.

Transformação de Unidades e Modelagem: Implementação de fórmulas de conversão (Temperatura) e modelos preditivos simples (Cálculo do Peso Ideal e Salário).

🛠️ Especificações Técnicas
Ambiente: De preferência, utilize o Google Colab ou o Jupyter Notebook para garantir a reprodutibilidade de dois experimentos.

Documentação: Utilize células de texto (Markdown) para explicar brevemente a lógica aplicada em cálculos mais complexos (como conversões térmicas).

Regras práticas: Os nomes variam de acordo com o contexto (ex: temp_celsiusem vez de apenas x), facilitando a leitura por outros membros de uma equipe de dados.
